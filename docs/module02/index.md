# Module 02 — Fluid Power Components

> **The machine:** the **Precision Hydraulic Lift Platform** — raises a load to a commanded height and holds it to within ±1 mm.
> **What this module does:** Module 01 taught you to *read* any hydraulic machine. Module 02 starts *building* this one — component by component.

---

## Where you are in the build

In Module 01 you established the platform's physics: it lifts two tonnes at 100 bar through a 50 mm bore (force = pressure × area), the energy travels from a power unit down a line to the actuator, and the actuator is a linear cylinder. Now you select and size the real parts.

Each lesson takes one component, states the decision it forces, and settles it with the platform's own numbers:

1. **The hydraulic cylinder** — the actuator that lifts. *Decision: stroke and acting type.*
2. **The power unit** — the pump, motor, and reservoir that make the flow. *Decision: pump size for the speed needed.*
3. **Lines, fittings, and ports** — the conductors that carry the fluid. *Decision: hose and port sizing.*
4. **The complete component set** — assembling the parts into a working platform. *Decision: the platform's component list and how it connects.*

By the end you will have a specified, connected set of components for the platform — ready for Module 03 to fill with the right fluid.
